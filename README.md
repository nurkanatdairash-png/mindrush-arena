# MindRush Arena

Brain-training gaming platform — reflex, memory, logic, strategy mini-games.

## Stack
- Vanilla JS + HTML/CSS (no framework)
- Supabase (auth + database + realtime)
- Service Worker (offline support / PWA)
- GitHub Pages

## Setup
1. Run `supabase-setup.sql` in your Supabase SQL editor
2. In Supabase → Authentication → Providers → Email → disable **Confirm email**
3. Deploy to GitHub Pages

## Play
- Guests can play all 4 games immediately — no account needed
- Register to save scores, earn XP, climb ranks, unlock achievements
